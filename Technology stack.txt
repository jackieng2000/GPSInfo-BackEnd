System Function
Data Model: A GPSLocation model with fields like latitude, longitude, and timestamp, serialized to JSON.

The frontend is on mobile phone. 
The GPSLocation will be stored every 15sec in localStorage by standard map component and 
the data will be send to backend through django restful api gateway every 15 sec for recording
Another function will call the backend to send out the latest position of all users.
